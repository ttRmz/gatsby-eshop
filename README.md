> A simple eshop starter to get up with Gatsby

## Install

```sh
# install dependencies

yarn
```

```sh
# instal gatsby cli globally

npm install -g gatsby-cli
```

## Usage

```sh
gatsby develop
```

## Run tests

```sh
yarn run test
```
